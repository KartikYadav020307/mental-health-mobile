# Agent Skill: Prompt Enhancer

**Trigger:** When the user asks you to "Enhance this prompt:" or "Improve this prompt:" followed by a basic request.

**Workflow:**
1. **Pause:** Do not write code yet.
2. **Analyze:** Read the user's basic request.
3. **Upgrade:** Rewrite their request into a highly specific, senior-level React Native technical mandate.
4. **Inject Specs:** Include architectural constraints, state management best practices, and refer to `DESIGN.md` for visual guidelines. Ensure it specifies `@expo/vector-icons` for iconography.
5. **Output:** Present the Enhanced Prompt back to the user and ask: "Here is the professional prompt. Should I execute this?"
